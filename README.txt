ЧТО ЭТО?
─────────
Baby Keyboard — полноэкранное приложение для маленьких детей.
Ребёнок может безопасно нажимать любые клавиши и водить мышкой,
не боясь сломать что-то на компьютере.

  - Нажатие клавиш — на экране появляются крупные буквы и цифры
  - Движение мыши — оставляет красивый радужный шлейф
  - Нежно-голубой фон с декоративными пузырьками

БЕЗОПАСНОСТЬ
────────────
Приложение блокирует системные комбинации клавиш:

  Windows: Alt+Tab, Alt+F4, Win, Ctrl+Esc
  macOS:   Cmd+Tab, Cmd+Q, Cmd+W, Cmd+H, Cmd+Space

Ребёнок не сможет случайно переключиться на другое приложение,
закрыть окно или открыть меню Пуск.

КАК ВЫЙТИ
──────────

    *** Нажмите  Ctrl + Enter  для закрытия приложения ***

Это единственный способ закрыть программу.
Малыш не сможет случайно нажать эту комбинацию.

ФАЙЛЫ
─────
  BabyKeyboard.exe   — версия для Windows
  BabyKeyboard       — версия для macOS (собрать через build_macos.sh)

ЗАПУСК
──────
  Windows:  Дважды кликните на BabyKeyboard.exe
  macOS:    Откройте терминал и выполните ./BabyKeyboard
            (предварительно: chmod +x BabyKeyboard)

ТРЕБОВАНИЯ ДЛЯ macOS
─────────────────────
При первом запуске macOS попросит дать разрешение в:
  Системные настройки → Конфиденциальность → Универсальный доступ

Без этого разрешения системные клавиши не будут заблокированы,
но само приложение будет работать.

СБОРКА macOS-ВЕРСИИ
───────────────────
На компьютере с macOS выполните:

  pip3 install pygame pyobjc-framework-Quartz pyinstaller
  bash build_macos.sh

Исполняемый файл появится в папке dist/
